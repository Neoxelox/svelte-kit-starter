# svelte-kit-starter

Full **SvelteKit** **TypeScript** starter featuring:

- SvelteKit
- Vite
- TypeScript
- TailwindCSS
- SCSS
- Autoprefixer
- JS & CSS Minification
- Enhanced Images & Fonts
- ESLint
- Prettier
- Stylelint
- Yarn

## develop

**`yarn dev`**

## lint

**`yarn lint`**

## format

**`yarn format`**

## build

**`yarn build`**

## preview

**`yarn preview`**
